we're thrilled to announce the release of rainbow (rnbw) version 0.1! it's the first major milestone on our journey to revolutionizing web development and design.

here are the exciting new features that you can now enjoy:

  • files/nodes api: this fundamental architecture offers a way to interact with files and html elements in a uniform and intuitive way.
  • tree view: easily visualize and navigate your project structure with a clean, easy-to-understand tree view.
  • code view: get into the nitty-gritty of your code with our code view, which provides a real-time perspective of your project's html, css, and javascript.
  • stage view: witness your designs come alive in the stage view. this live, editable representation of your project showcases the changes you make in real-time.
  • zero-sync: with rnbw's zero-sync approach, your edits are instantly reflected across tree view, code view, and stage view. no more manual refreshes or updates.
  • command menus: achieve more with less effort using our dynamic command menus. whether you're adding new components or initiating complex actions, our command menus streamline your workflows.

we're just getting started and can't wait to see what you'll create with rnbw v0.1! your feedback is crucial for our next steps, so don't hesitate to share your thoughts and experiences.

stay tuned for more updates as we continue to build a more powerful, intuitive, and efficient tool for web design and development.

thanks for being part of our journey!
